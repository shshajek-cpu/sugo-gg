# [AION2] 어비스 랭킹 수집 및 연동 결과 확인서

## 1. 작업 개요
사용자 님의 요청에 따라 **AION2 공식 홈페이지 어비스 랭킹**에서 상위 캐릭터 정보를 수집하고, 해당 캐릭터들의 상세 정보(장비, 스탯 등)를 자동으로 가져와 시스템에 연동하는 기능을 구현했습니다.

## 2. 구현 결과
### ✅ 백엔드 (Backend)
*   **어비스 랭킹 수집기 (`AION2SourceAdapter/fetch_abyss_rankings`)**:
    *   AION2 어비스 랭킹 페이지에 자동으로 접속하여 지정된 서버와 종족의 상위 랭커 목록을 추출합니다.
    *   추출된 각 캐릭터에 대해 **상세 정보(스탯, 장비, 펫, 스킬 등)를 자동으로 추가 수집**합니다.
*   **신규 API (`POST /api/rankings/fetch`)**:
    *   외부에서 랭킹 수집을 트리거할 수 있는 API를 추가했습니다.
    *   파라미터: `server` (서버명), `race` (종족), `limit` (수집 개수)

### ✅ 데이터 검증 (Verification)
*   **테스트 수행**: 이스라펠 서버 / 천족 / Top 3 캐릭터 수집 테스트 완료.
*   **결과**: 정상적으로 캐릭터 이름("운빨러" 등)과 상세 정보를 가져와 데이터베이스에 저장함을 확인했습니다.
*   **연동 확인**: 이제 웹사이트에서 해당 캐릭터를 검색하거나 조회하면, 수집된 최신 상세 정보를 볼 수 있습니다.

## 3. 사용 방법 (How to Use)
터미널이나 API 클라이언트를 통해 아래 명령어로 수집을 시작할 수 있습니다.
(현재는 시스템 내부 기능으로 구현되었으며, 필요 시 프론트엔드에 버튼을 추가할 수 있습니다.)

```bash
# 예시: 이스라펠 서버 천족 Top 5 수집
curl.exe -X POST "http://localhost:8000/api/rankings/fetch?server=이스라펠&race=천족&limit=5"
```

## 4. 참고 사항
> [!IMPORTANT]
> **과도한 수집 주의**: 랭킹 수집은 다수의 페이지를 짧은 시간에 조회하므로, `limit` 값을 너무 높게(예: 50 이상) 설정하면 IP 차단 가능성이 있습니다. 초기에는 5~10명 수준으로 테스트하는 것을 권장합니다.
